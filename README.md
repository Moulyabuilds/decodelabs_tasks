# HelloBot - Rule-Based Greeting Chatbot

## Description
HelloBot is a simple rule-based AI chatbot developed using Python. It responds to predefined user inputs such as greetings and exits using if-else statements.

## Features
- Greets users
- Responds to basic inputs
- Handles exit commands
- Runs continuously until the user exits

## Technologies Used
- Python

## How to Run
1. Download the project.
2. Open a terminal in the project folder.
3. Run:
   python chatbot.py
